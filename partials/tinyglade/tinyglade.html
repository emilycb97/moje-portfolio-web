Projekt: Tajemnicze opactwo (Tiny Glade) Project: Mysterious abbey (Tiny Glade)

Zadaniem projektowym było stworzenie koncepcji Tajemniczego Opactwa przy użyciu narzędzia Tiny Glade. Projekt skupia się na budowaniu klimatu poprzez architekturę i oświetlenie, co stanowi element moich prac nad rozwojem w kierunku Environment Designu. Poniżej znajduje się pełna dokumentacja wizualna projektu.

The project task was to create a concept of a Mysterious Abbey using the Tiny Glade tool. The project focuses on building mood through architecture and lighting, which is part of my work on developing towards Environment Design. Below is the full visual documentation of the project.
