Generator fraktali (Processing) Fractal generator (Processing)

Projekt z kategorii Creative Coding napisany w języku Java przy użyciu środowiska Processing. Jest to interaktywny generator pozwala na generowanie i manipulację strukturami rekursywnymi (Drzewo Fraktalne oraz Trójkąt Sierpińskiego). Kod źródłowy jest renderowany w czasie rzeczywistym bezpośrednio w przeglądarce dzięki technologii WebGL/Canvas.

A Creative Coding project written in Java using the Processing environment. It is an interactive generator that allows the generation and manipulation of recursive structures (Fractal Tree and Sierpinski Triangle). The source code is rendered in real-time directly in the browser thanks to WebGL/Canvas technology.

Jak sterować:
Lewy Klik Myszy - Zmiana fraktala (Drzewo / Sierpiński)
Strzałki GÓRA/DÓŁ - Zmiana głębokości rekursji
R - Reset
How to control:
Left Mouse Click - Change fractal (Tree / Sierpinski)
UP/DOWN Arrows - Change recursion depth
R - Reset
