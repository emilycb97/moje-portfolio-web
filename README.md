# Portfolio Osobiste - Emilia Szczerba

Interaktywna strona internetowa pełniąca rolę cyfrowego portfolio. Projekt prezentuje moje wszechstronne umiejętności z zakresu grafiki 3D, projektowania UI/UX, wektorowej grafiki SVG oraz web developmentu. Strona posiada pełne wsparcie dla dwóch języków i pozwala na interaktywne przeglądanie projektów.

## Funkcjonalności 
* **Dwujęzyczność (PL/EN):** Dynamiczna zmiana języka całej strony bez przeładowywania widoku, obsługiwana przez autorski skrypt. Wybór użytkownika jest zapamiętywany dzięki `localStorage`.
* **Interaktywne modele 3D:** Osadzone w przeglądarce przestrzenne rzeźby i obiekty (w formacie `.glb`), wyposażone w pełne sterowanie kamerą (obrót, zoom) oraz tryb pełnoekranowy (Fullscreen API).
* **Autorska galeria (Lightbox):** Zaimplementowana od podstaw przeglądarka zrzutów ekranu i grafik, wyposażona w płynną nawigację kliknięciami oraz z użyciem klawiatury (strzałki, escape).
* **Responsywny interfejs:** Elastyczny układ siatki (Grid/Flexbox) w estetyce nawiązującej do stylu hakerskiego i cyberpunkowego (ciemne tło, zielone akcenty, font *Oxanium*).

## Technologie 
* **Języki:** HTML5, CSS3, Vanilla JavaScript
* **Biblioteki/Komponenty:** `<model-viewer>` (do renderowania modeli 3D w czasie rzeczywistym)
* **Formaty danych:** GLB, SVG, XML/XSLT

*Data realizacji: Kwiecień 2026*

# Personal Portfolio - Emilia Szczerba 

An interactive website acting as a digital portfolio. The project showcases my versatile skills in 3D graphics, UI/UX design, SVG vector graphics, and web development. The site features full bilingual support and allows for the interactive viewing of projects directly in the browser.

## Features 
* **Bilingual Support (PL/EN):** Dynamic language switching without page reloads, driven by a custom script. The user's choice is remembered across sessions using `localStorage`.
* **Interactive 3D Models:** Embedded spatial sculptures and objects (in `.glb` format), featuring full camera controls (rotation, zoom) and a fullscreen mode (Fullscreen API).
* **Custom Lightbox Gallery:** A built-from-scratch image viewer for screenshots and graphics, equipped with smooth click navigation and keyboard support (arrows, escape).
* **Responsive Interface:** A flexible grid layout (Grid/Flexbox) styled with a hacker/cyberpunk aesthetic (dark background, green accents, *Oxanium* font).

## Technologies
* **Languages:** HTML5, CSS3, Vanilla JavaScript
* **Libraries/Components:** `<model-viewer>` (for real-time 3D model rendering)
* **Data Formats:** GLB, SVG, XML/XSLT

*Developed: April 2026*
